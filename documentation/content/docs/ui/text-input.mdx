A controlled single-line text input with cursor navigation, inline editing, and render prop support.

While focused, TextInput handles all keyboard input. Tab, shift+tab, and Escape bubble to parent components normally. Enter bubbles unless `onSubmit` is provided.

```tsx title="Basic Usage"
import { TextInput } from 'giggles/ui';
import { useState } from 'react';

function NameInput() {
  const [value, setValue] = useState('');

  return <TextInput label="Name:" value={value} onChange={setValue} placeholder="John Doe" />;
}
```

```tsx title="Form with Submit"
import { TextInput } from 'giggles/ui';
import { FocusGroup } from 'giggles';
import { useState } from 'react';

function SearchForm() {
  const [query, setQuery] = useState('');

  return (
    <FocusGroup direction="vertical">
      <TextInput
        label="Search:"
        value={query}
        onChange={setQuery}
        onSubmit={(value) => performSearch(value)}
        placeholder="type to search..."
      />
    </FocusGroup>
  );
}
```

### Keybindings

| Key | Action |
| --- | --- |
| Characters | Insert at cursor position |
| `←` `→` | Move cursor |
| `Home` `End` | Jump to start/end |
| `Backspace` | Delete character before cursor |
| `Delete` | Delete character at cursor |
| `Tab` / `Shift+Tab` | Move to next/previous field |
| `Enter` | Fire `onSubmit` if provided, otherwise ignored |
| `Escape` | Ignored (available for parent keybindings) |

### TextInputRenderProps

The `render` prop receives pre-split text segments around the cursor.

```tsx title="Custom Render"
<TextInput
  label="Notes:"
  value={value}
  onChange={setValue}
  render={({ before, cursorChar, after, focused }) => (
    <Text color={focused ? 'cyan' : 'white'}>
      {before}
      <Text inverse>{cursorChar}</Text>
      {after}
    </Text>
  )}
/>
```
