A scroll position indicator for windowed lists. Supports three display styles: arrows, scrollbar, and counter.

Paginator is a pure display component — it has no focus or interactivity. Pass it the total item count, current scroll offset, and number of visible items. It computes and renders the appropriate indicator. VirtualList uses Paginator internally, but you can also use it directly in custom scrollable layouts.

When the total number of items fits within the visible window, Paginator renders nothing.

```tsx title="Standalone Usage"
import { Paginator } from 'giggles/ui';

<Paginator total={100} offset={20} visible={10} style="scrollbar" />
```

### Arrows

Shows `↑` and `↓` indicators above and below the list when there are items outside the visible window.

<Terminal>
  <PaginatorArrowsExample />
</Terminal>

### Scrollbar

Shows a vertical bar beside the list with a thumb that tracks scroll position.

<Terminal>
  <PaginatorScrollbarExample />
</Terminal>

### Counter

Shows a text range like `1–10 of 50` below the list.

  <PaginatorCounterExample />
</Terminal>
