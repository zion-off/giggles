A yes/no prompt. Press `y` to confirm, `n` to decline, or Enter to accept the default.

The `defaultValue` prop controls which option is pre-selected. The hint text updates to reflect this â€” `(Y/n)` when defaulting to yes, `(y/N)` when defaulting to no. Unlike Select, Confirm is a one-shot interaction: `onSubmit` fires once with a boolean and there's no controlled value state.

```tsx title="Basic Usage"
import { Confirm } from 'giggles/ui';

function DeletePrompt() {
  return (
    <Confirm
      message="Delete this file?"
      onSubmit={(yes) => {
        if (yes) deleteFile();
      }}
    />
  );
}
```

### Confirm

<TypeTable

### Keybindings

| Key | Action |
| --- | --- |
| `y` | Confirm (yes) |
| `n` | Decline (no) |
| `Enter` | Accept default |
| `Tab` / `Shift+Tab` | Move to next/previous component |
