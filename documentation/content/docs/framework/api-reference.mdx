## GigglesProvider

Root provider for giggles applications. Wraps your app in the necessary context providers and sets up the input router.

```tsx title="Basic Setup"
import { render } from 'ink';
import { GigglesProvider } from 'giggles';

render(
  <GigglesProvider>
    <App />
  </GigglesProvider>
);
```
